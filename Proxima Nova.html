Proxima Nova

This was a university based project where we were to create a text only poster for a famous typeface. The poster was to reflect the personality of the typeface as well as provide relevant information about the history, use and key characteristics. In creating my poster solution, I used the literal meaning of proxima and nova to create a constellation like poster artwork using only glyphs from the font family and the colours of the night sky. This solution was created on Adobe Illustrator.
